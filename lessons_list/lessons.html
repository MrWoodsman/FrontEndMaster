krótko o html

HTML (HyperText Markup Language) to język służący do tworzenia stron internetowych. Zawiera znaczniki, które określają różne elementy na stronie, takie jak nagłówki, akapity, listy, tabele itp. Znaczniki są umieszczane w nawiasach klamrowych i składają się z nazwy oraz atrybutów. HTML jest podstawą do tworzenia stron internetowych i może być używany z innymi technologiami, takimi jak CSS i JavaScript, aby dostosować wygląd i działanie strony.
